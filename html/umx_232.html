umx_open R Documentation

Open a file or folder

Description

Open a file or folder. Works on OS X, mostly on windows, and hopefully on unix.

Usage

umx_open(filepath = getwd())

Arguments

filepath

The file to open

Details

NOTE: Your filepath is shQuote()'d by this function.

Value

None

References
