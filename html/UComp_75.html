sumStats R Documentation

sumStats

Description

Summary statistics of a matrix of variables

Usage

sumStats(y, decimals = 5)

Arguments

y

a vector, matrix of time series
decimals

number of decimals for table

Details

Position, dispersion, skewness, kurtosis, etc.

Value

Table of values
