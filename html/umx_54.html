reliability R Documentation

Report coefficient alpha (reliability)

Description

Compute and report Coefficient alpha (extracted from Rcmdr to avoid its dependencies)

Usage

reliability(S)

Arguments

S

A square, symmetric, numeric covariance matrix

Value

None
