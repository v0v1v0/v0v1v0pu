differences R Documentation

Specifying the order of differences

Description

This function is used to specify that the time series will be preprocessed using first differences.

Usage

differences(n = -1)

Arguments

n

An integer specifying the order of differences to be applied. If the default (-1) is used, the order of differences needed by the time series will be computed by the forecast::ndiffs() function.

Value

An integer with the order of differences to be applied.

Examples

differences(1)
