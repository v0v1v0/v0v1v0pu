bangladesh R Documentation

Bangladesh 1974

Description

The data gathered by the 1974 Bangladesh Retrospective Survey of Fertility and Mortality can be used to demonstrate the estimation of child mortality from summary birth histories using the Trussell version of the BRASS method and the Coale-Demeny model life tables coale_demeny_ltm.

Usage

data(bangladesh)

Format

A data frame
