convert_marginal_tax_rates R Documentation

Recode marginal tax rates.

Description

Marginal tax rates are specified with the marginal_tax_rates parameter. The possible values are descriptive strings. But,TAXSIM requires integers. Convert descriptive strings to integers.

Usage

convert_marginal_tax_rates(marginal_tax_rate)

Arguments

marginal_tax_rate

String representing the marginal_tax_rate parameter in taxsim_calculate_taxes
