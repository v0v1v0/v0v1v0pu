outersect R Documentation

Outersect

Description

Performs set outersection on two vectors. The opposite operation from intersect!

Usage

outersect(x, y)

Arguments

x, y

vectors

Value

a vector of the non-common elements of x and y.

See Also

intersect

Examples

x = 1:10
y = 2:11

# c(1,11)
outersect(x,y)
