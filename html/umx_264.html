umx_set_separator R Documentation

Set the separator

Description

Set umx_default_separator (used in CI\[low sep high\] ). Default = ","

Usage

umx_set_separator(umx_default_separator = NULL, silent = FALSE)

Arguments

umx_default_separator

separator for CIs etc. (if empty, returns the current value)
silent

If TRUE, no message will be printed.

Value

- Current umx_default_separator

See Also

Other Get and set: umx_get_checkpoint(), umx_get_options(), umx_set_auto_plot(), umx_set_auto_run(), umx_set_checkpoint(), umx_set_condensed_slots(), umx_set_cores(), umx_set_data_variance_check(), umx_set_dollar_symbol(), umx_set_optimization_options(), umx_set_optimizer(), umx_set_plot_file_suffix(), umx_set_plot_format(), umx_set_plot_use_hrbrthemes(), umx_set_silent(), umx_set_table_format(), umx

Examples

library(umx)
umx_set_separator() # show current state
old = umx_set_separator(silent=TRUE) # store existing value
umx_set_separator("|")
umxAPA(.3, .2)
umx_set_separator(old)    # reinstate
