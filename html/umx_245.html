umx_rot R Documentation

Rotate a vector

Description

umx_rot rotates the items of a vector (1 place, by default). So: c(1,2,3) -> c(2,3,1)

Usage

umx_rot(vec, na.last = FALSE)

Arguments

vec

vector to rotate
na.last

Whether to set the last value to NA (default = FALSE)

Value

  • mxModel()

References

  • https://tbates.github.io

Examples

umx_rot(1:10)
umx_rot(c(3,4,5,6,7))
# [1] 4 5 6 7 3
