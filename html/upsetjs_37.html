setQueries R Documentation

set the queries

Description

set the queries

Usage

setQueries(upsetjs, queries = list())

Arguments

upsetjs

an object of class upsetjs or upsetjs_proxy
queries

the queries to set

Value

the object given as first argument

Examples

upsetjs() %>%
  fromList(list(a = c(1, 2, 3), b = c(2, 3))) %>%
  setQueries(list(list(name = "Q1", color = "red", set = "b")))
