bind_iterations R Documentation

Bind Iterations

Description

Combines two MCMC objects (with the same parameters and chains) by iterations.

Usage

bind_iterations(x, x2, ...)

Arguments

x

An object.
x2

A second object.
...

Other arguments passed to methods.

Value

The combined object.
