vplayout

Description

Viewport

Usage

vplayout(x, y)

Arguments

x

The x cell of the viewport to push into.
y

The y cell of the viewport to push into.

Details

Creates viewport for pushing ggplot objects to parts of a console.

Value

An R object of class viewport.
