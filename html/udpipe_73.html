txt_overlap R Documentation

Get the overlap between 2 vectors

Description

Get the overlap between 2 vectors

Usage

txt_overlap(x, y)

Arguments

x

a vector
y

a vector

Value

a vector with elements of x which are also found in y

Examples

x <- c("a", "b", "c")
y <- c("b", "c", "e", "z")
txt_overlap(x, y)
txt_overlap(y, x)
