panama R Documentation

Panama 1976

Description

The data gathered by a survey in Panama between August and October 1976 can be used to demonstrate the estimation of child mortality from summary birth histories using the Trussell version of the BRASS method and the Coale-Demeny model life tables coale_demeny_ltm.

Usage

data(panama)

Format

A data frame
