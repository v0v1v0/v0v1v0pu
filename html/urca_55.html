show.urca R Documentation

Function to show objects of classes for unit root tests

Description

The function show.urca is called within the defined methods for classes ca.jo, cajo.test, ca.po, ur.df, ur.ers, ur.kpss, ur.po, ur.pp, ur.df, ur.sp and ur.za.

Usage

show.urca(object)

Arguments

object

Object of class contained in urca.

Details

This function is called by method show.

Value

The Name and test statistic of a unit root/cointegration test.
