Description

Plot acf objects

Usage

## S3 method for class 'acf'
autoplot(object, xlab = x, ylab = y,
  title = sprintf("%s Plot", y), ...)

Arguments

object

An acf object.
xlab

X-axis label.
ylab

y-axis label.
title

Graph title.
...

Further arguments.

Details

Plot acf (and pacf) objects.

Value

A ggplot object.

Author(s)

Jared P. Lander

Examples


autoplot(acf(sunspot.year, plot=FALSE))
autoplot(pacf(sunspot.year, plot=FALSE))
