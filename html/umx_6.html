extractAIC.MxModel R Documentation

Extract AIC from MxModel

Description

Returns the AIC for an OpenMx model. Original Author: Brandmaier

Usage

## S3 method for class 'MxModel'
extractAIC(fit, scale, k, ...)

Arguments

fit

an fitted mxModel() from which to get the AIC
scale

not used
k

not used
...

any other parameters (not used)

Value

  • AIC value

References

  • https://openmx.ssri.psu.edu/thread/931#comment-4858

Examples

## Not run: 
require(umx)
data(demoOneFactor)
manifests = names(demoOneFactor)
m1 = umxRAM("One Factor", data = demoOneFactor, type = "cov",
	umxPath("G", to = manifests),
	umxPath(var = manifests),
	umxPath(var = "G", fixedAt = 1)
)
extractAIC(m1)
# -2.615998
AIC(m1)

## End(Not run)
