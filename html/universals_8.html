converged_terms R Documentation

Converged Terms

Description

Tests whether each term of an object has converged.

Usage

converged_terms(x, ...)

Arguments

x

An object.
...

Other arguments passed to methods.

Value

A list of uniquely named logical objects with whether each term has converged.
