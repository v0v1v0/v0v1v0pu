ETSvalidate R Documentation

ETSvalidate

Description

Shows a table of estimation and diagnostics results for ETS models

Usage

ETSvalidate(m)

Arguments

m

an object of type ETS created with ETSmodel

Value

The same input object with the appropriate fields filled in, in particular:

table

Estimation and validation table

Author(s)

Diego J. Pedregal

See Also

ETS, ETSmodel, ETSvalidate, ETScomponents

Examples

## Not run: 
m1 <- ETSmodel(log(gdp))
m1 <- ETSvalidate(m1)

## End(Not run)
