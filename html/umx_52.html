qm R Documentation

qm

Description

Quickmatrix function

Usage

qm(..., rowMarker = "|")

Arguments

...

the components of your matrix
rowMarker

mark the end of each row

Value

- matrix

# simple example
qm(0, 1 |
   2, NA)
